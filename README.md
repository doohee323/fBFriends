fBFriends
=========

### [Demo](http://fbfriend.herokuapp.com/)

## Install

```shell
git clone https://github.com/doohee323/fBFriends.git
cd fBFriends
npm install
bower install
grunt dist
```

## Run

```shell
grunt serve
```

## skills
angular.js, facebook api(oauth)

### Usage
0. add domain into local computer's hosts file for facebook oauth
	vi /etc/hosts
	127.0.0.1 topzone.dyndns.org

1. open url : http://topzone.dyndns.org:9000/
2. Push the facebook button.
3. Select one of them.
4. Guess this friend's last name. and click the picture. 
5. Retrieve the result.
